TVP Frontend
=================

This is an Angular frontend scaffold that integrates with the Auth and Compras services.

Prerequisites
- Node.js (16+ recommended) and npm installed.

Install and run
```bash
cd front
npm install
npm run start
```

Configuration
- Edit `src/environments/environment.ts` to point `authServiceUrl` and `comprasServiceUrl`.

Notes
- The repo does not include node_modules. Use `npm install` locally.
